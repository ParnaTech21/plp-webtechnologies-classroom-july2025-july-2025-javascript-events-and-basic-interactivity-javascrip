# 🧠 Interactive Web Pages with JavaScript

Welcome! This project is a practical assignment on enhancing web interactivity using JavaScript. It guides you through building **dynamic features** and **validating forms**, making your web pages engaging and user-friendly.

---

## 🚀 Features

- **Event Handling:** Buttons, input fields, and links respond instantly to user actions.
- **Interactive Elements:**  
  - Light/Dark mode toggle  
  - Counter game  
  - Collapsible FAQ section  
- **Form Validation:**  
  - Custom JavaScript validation for name, email, and password  
  - Real-time error messages and feedback

---

## 📁 Files

- `index.html` — The main web page with interactive sections and the form.
- `script.js` — All event handling, interactive features, and validation logic.
- `style.css` — Visual styles for a clean, responsive look.

---

## 📝 How to Use

1. **Clone or download** the repository.
2. Open `index.html` in your browser.
3. Interact with the features:
   - Toggle between light and dark mode.
   - Play with the counter.
   - Reveal answers in the FAQ.
   - Fill out and submit the contact form—watch for validation feedback.

---

## 💡 Educational Outcomes

- Master event listeners and DOM manipulation.
- Build creative, interactive components.
- Implement robust client-side form validation.
- Write clear, modular, well-commented JavaScript code.
